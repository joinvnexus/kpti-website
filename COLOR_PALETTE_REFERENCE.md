## Color Usage Guidelines

### ‚úÖ DO's

- ‚úÖ Use **Primary Blue** for main CTAs and brand elements
- ‚úÖ Use **Secondary Teal** for alternative actions and highlights
- ‚úÖ Use **Accent Green** for success states and positive feedback
- ‚úÖ Use **Dark Navy** as dark mode background
- ‚úÖ Use **Off White** for cards and lighter backgrounds
- ‚úÖ Use **Muted Colors** for secondary text and disabled states
- ‚úÖ Ensure adequate contrast (4.5:1 minimum for body text)
- ‚úÖ Use semantic colors (red = danger, green = success, etc.)

### ‚ùå DON'Ts

- ‚ùå Don't use primary colors on dark backgrounds without additional contrast
- ‚ùå Don't use multiple primary colors for equal weight elements
- ‚ùå Don't change color on hover without maintaining readability
- ‚ùå Don't use pure black or pure white for text (use dark slate/off-white)
- ‚ùå Don't override the design system colors without approval
- ‚ùå Don't use colors for information-only communication (add icons/text)
